********
Glossary
********

.. glossary::

    JavaScript
        Programming language specified in the ECAMScript language specification.

        .. seealso:: http://www.w3schools.com/js/

    Node.js
        A JavaScript runtime built on Chrome's V8 JavasScript engine.

        .. seealso:: https://nodejs.org/

    NPM
        Package manager for :term:`Node.js`.

        .. seealso:: https://www.npmjs.com

    Python
        High-level programming language widely used for general purpose
        programming.

        .. seealso:: https://www.python.org/

    PyPI
        Python package index. A repository of software for the Python
        programming language.

        .. seealso:: https://pypi.python.org/pypi

    React
        A :term:`JavaScript` framework for build user interfaces.

        .. seealso:: https://facebook.github.io/react/

    Sphinx
        A tool that makes it easy to create intelligent and beautiful
        documentation.

        .. seealso:: http://www.sphinx-doc.org

    Virtualenv
        A tool to create isolated Python environments.

        .. seealso:: https://virtualenv.pypa.io/en/latest/
